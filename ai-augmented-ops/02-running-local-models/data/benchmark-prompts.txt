# Security benchmark prompts for local model evaluation
# Format: one prompt per block, separated by ---
# These represent realistic SOC analyst queries — short enough to run in < 60 s on CPU.

PROMPT_1
You are a security analyst. A Windows endpoint generated this alert: "Process cmd.exe spawned by winword.exe with argument: /c powershell -EncodedCommand ZQBjAGgAbwAgAGgAZQBsAGwAbwA=". In one sentence, identify the ATT&CK technique and recommended immediate action.

---

PROMPT_2
You are a threat intelligence analyst. Summarise CVE-2021-44228 (Log4Shell) in three bullet points: what it is, what systems are affected, and the primary detection method. Be factual and concise.

---

PROMPT_3
You are a SOC analyst. An endpoint security tool generated this alert: "Unusual outbound connection from host MERIDIAN-FIN-07 to 185.220.101.42:443 at 02:14 UTC. Process: svchost.exe. Duration: 47 minutes." List three hypotheses (ranked by likelihood) and the first log source you would pivot to for each.

---

PROMPT_4
You are a malware analyst. A file has SHA-256 hash e3b0c44298fc1c149afbf4c8996fb92427ae41e4649b934ca495991b7852b855. What does this hash represent and what does it tell you about the file?

---

PROMPT_5
You are a security engineer. Write a single Sigma rule (YAML) that detects PowerShell launching with the -EncodedCommand flag on a Windows host. Include the logsource, detection, and condition fields. Tag it with the correct MITRE ATT&CK technique ID.
